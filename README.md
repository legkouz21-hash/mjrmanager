# MJRManager

**MJRManager** — мощный инструмент для работы с JAR-архивами: декомпиляция, редактирование и перекомпиляция Java-классов с современным графическим интерфейсом.

![Java](https://img.shields.io/badge/Java-17-orange)
![JavaFX](https://img.shields.io/badge/JavaFX-21.0.6-blue)
![License](https://img.shields.io/badge/license-MIT-green)

## 🚀 Возможности

- **📦 Открытие JAR/WAR/EAR архивов** — просмотр структуры файлов в удобном древовидном виде
- **🔍 Декомпиляция .class файлов** — автоматическая декомпиляция с использованием CFR Decompiler
- **✏️ Редактирование исходного кода** — встроенный редактор с подсветкой синтаксиса Java
- **⚙️ Перекомпиляция классов** — два компилятора на выбор:
  - **Eclipse JDT** (мягкий режим) — компилирует даже с ошибками
  - **javac** (строгий режим) — стандартный Java-компилятор
- **💾 Сохранение изменений** — обновление JAR-архива с автоматическим созданием резервной копии
- **🔎 Поиск по файлам** — быстрый поиск классов в дереве проекта
- **📜 История файлов** — список недавно открытых JAR-архивов
- **🐛 Консоль отладки** — встроенный логгер для отслеживания операций
- **👁️ Просмотр байткода** — возможность просмотра hex-дампа .class файлов

## 🔧 Установка и запуск

### Клонирование репозитория

```bash
git clone https://github.com/legkouz21-hash/mjrmanager.git
cd mjrmanager
```

### Сборка проекта

#### Windows:
```bash
./mvnw clean package
```

#### Linux/macOS:
```bash
chmod +x mvnw
./mvnw clean package
```

### Запуск приложения

```bash
./mvnw javafx:run
```

## 📦 Создание исполняемого файла

Для создания standalone-приложения:

```bash
./mvnw javafx:jlink
```

Исполняемый файл будет создан в директории `target/app/`.

## 🎯 Использование

1. **Открытие JAR-файла**
   - Меню: `Файл → Открыть JAR`
   - Или используйте список недавних файлов

2. **Декомпиляция класса**
   - Выберите `.class` файл в дереве проекта
   - Класс автоматически декомпилируется и откроется в редакторе

3. **Редактирование кода**
   - Измените код в редакторе
   - Используйте подсветку синтаксиса для удобства

4. **Компиляция изменений**
   - Нажмите кнопку `Компилировать` или `Ctrl+B`
   - Выберите компилятор в меню `Настройки → Компилятор`

5. **Сохранение JAR**
   - `Файл → Сохранить` — перезаписать оригинальный файл (создаётся backup)
   - `Файл → Сохранить как...` — сохранить в новый файл

## 🎨 Особенности интерфейса

- **Древовидная структура** — навигация по содержимому JAR
- **Вкладки редактора** — работа с несколькими классами одновременно
- **Нумерация строк** — удобная навигация по коду
- **Контекстное меню** — просмотр исходного кода или байткода
- **Статус-бар** — отображение текущих операций
- **Прогресс-бар** — индикация фоновых задач
