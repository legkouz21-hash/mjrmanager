# Changelog

## Что добавлено

### ✨ 4 новые функции

1. **🔍 Глобальный поиск** (`Ctrl+Shift+F`)
   - Поиск текста во всех классах JAR
   - Поддержка regex и учета регистра

2. **📄 Редактор манифеста**
   - Просмотр и редактирование META-INF/MANIFEST.MF
   - Кнопка "Манифест" на панели

3. **➕ Добавление/удаление файлов**
   - Меню "JAR" → "Добавить файлы..."
   - Меню "JAR" → "Удалить выбранный файл"

4. **📦 Экспорт в исходники** (`Ctrl+E`)
   - Декомпиляция всех классов в .java файлы
   - Сохранение структуры проекта
